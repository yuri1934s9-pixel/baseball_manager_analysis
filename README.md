# Baseball Manager Analysis (プロ野球監督評価分析パイプライン)

本プロジェクトは、プロ野球の歴史的データから「監督の采配効果」を定量化し、その要因および市場価値（年俸や契約年数）への影響を検証するためのエンドツーエンドのデータ分析パイプラインです。

データの自動収集（スクレイピング）から、セイバーメトリクス指標を用いたパネルデータの作成、そして計量経済学的手法を用いた回帰分析までの一連のプロセスを完全に再現することができます。

## 全体の分析フロー

本プロジェクトは、大きく分けて「データ準備」と「実証分析」の2つのフェーズ（計4ステップ）で構成されています。

### フェーズ 1: データ準備・前処理

#### 1. データ収集とテキスト抽出 (`scrape_baseball_data.R`)
- **スクレイピング**: 日本プロ野球記録データベース（2689web.com）から1950年〜2025年までの年度別HTMLファイルを自動ダウンロード（サーバー負荷軽減のため `Sys.sleep(1)` を設定）。
- **テキスト抽出**: 各年度のHTMLから野手・投手の成績テーブルを解析し、球団名を正規化した上でテキスト形式（`.txt`）として抽出し保存します。

#### 2. 特定指標のCSV化 (`extract_metrics.R`)
- 前ステップで生成されたテキスト群から、分析対象とする特定の指標（例: 「打席」「防率」など）を自動判別して抽出。
- 年度ごとのデータを横持ち（ワイド型）のCSVに整形し、指定のディレクトリへ出力します。

#### 3. パネルデータの結合・クリーニング (`process_panel_data.R`)
- 選手データと監督・コーチのデータを読み込み、縦持ち（ロング型）に変換した上で、チームと年度をキーにして結合（`full_join`）します。
- 分析に適したクロスセクション・時系列のパネルデータを構築し、欠損値（NA）の処理を行った上で最終的な分析用CSVを出力します。

### フェーズ 2: 実証分析

#### 4. 采配効果の抽出 (`manager_effect.R`)
- チーム勝率を目的変数とし、戦力要因や環境要因をコントロールした回帰分析を行います。
  - **戦力指標の高度化**: wOBA, wSB, FIP, DER といった高度なセイバーメトリクス指標によりモデルを制御。
  - **環境要因の制御**: リーグ特性、Aクラス経験、年度ごとの固定効果（Fixed Effects）を考慮。
  - **監督効果の算出**: 監督ごとのダミー変数から得られた係数を、最小値を0とした相対的なスコアに変換して指標化。

#### 5. 采配効果の要因および影響分析 (`regression.R`)
- 算出された「監督スコア」が何に起因し、市場でどう評価されているかを多角的に分析します。
  - **要因分析**: 現役時の守備ポジション、監督経験の有無、外国人ダミーが采配効果に与える影響を検討。
  - **市場価値分析**: 采配効果が初期年俸（対数モデル）や契約年数（負の二項回帰モデル）にどう反映されているかを検証。
  - **頑健性の確認**: 球団ごとのクラスター標準誤差（`lm_robust`）を用いた推定。
