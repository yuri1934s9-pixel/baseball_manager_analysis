分析のフロー
本研究は、大きく分けて以下の2つのフェーズで構成されています。

1. 采配効果の抽出 (manager_effect.R)
チーム勝率を目的変数とし、選手の能力（戦力要因）や環境要因をコントロールした回帰分析を行います。

戦力指標の高度化: 打率・本塁打などの伝統的指標から、wOBA, wSB, FIP, DER といった高度なセイバーメトリクス指標へモデルを改良。

環境要因の制御: リーグ特性、Aクラス経験、年度ごとの固定効果（Fixed Effects）を考慮。

監督効果の算出: 監督ごとのダミー変数から得られた係数を、相対値（最小値を0としたスコア）に変換して指標化。

2. 采配効果の要因および影響分析 (regression.R)
算出された「監督効果」が何に起因し、どう評価されているかを分析します。

要因分析: 現役時の守備ポジション、監督経験の有無、外国人ダミーが采配効果に与える影響を検討。

市場価値分析: 采配効果が初期年俸（対数モデル）や契約年数（負の二項回帰モデル）にどう反映されているかを検証。

頑健性の確認: 球団ごとのクラスター標準誤差（lm_robust）を用いた推定。

使用している主なRパッケージ
分析の再現には以下のライブラリが必要です。

estimatr: 強固な標準誤差を伴う線形推定

modelsummary: 分析結果の美しい表出力

marginaleffects: 負の二項回帰等の限界効果の算出

tidyverse (dplyr, ggplot2 等): データ前処理および可視化

MASS, AER: 過分散検定および負の二項回帰

実行方法
監督効果/ フォルダに必要なCSVデータ（パネルデータ等）を配置します。

manager_effect.R を実行し、監督ごとのスコアを算出します。

regression.R を実行し、要因分析の結果（推定結果/ フォルダに出力）を確認します。

*データは含んでおりません。
