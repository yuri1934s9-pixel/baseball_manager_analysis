# Baseball Manager Analysis (プロ野球監督評価分析)

本プロジェクトは、プロ野球監督の「采配効果」を定量化し、その要因と市場評価を分析するためのRスクリプト群です。セイバーメトリクスの指標を活用し、戦力要因を制御した上での純粋な監督の影響度を導き出します。

## 分析パイプライン
本研究は以下の2つのフェーズで構成されています。

### 1. 采配効果の算出 (`manager_effect.R`)
チームの勝率を目的変数とし、戦力要因と環境要因を制御した回帰分析を行い、監督ごとの「采配スコア」を算出します。
* **戦力指標の高度化**: wOBA, wSB, FIP, DER などのセイバーメトリクス指標を採用。
* **環境要因の制御**: リーグ特性、Aクラス経験、年度固定効果（Fixed Effects）の考慮。
* **スコア化**: 監督ダミー変数の係数を抽出し、相対スコア（最小値=0）に変換。

### 2. 要因および影響分析 (`regression.R`)
算出された「采配効果」が、監督の属性や市場価値にどう関連しているかを検証します。
* **要因分析**: 監督の現役時代ポジション、経験年数、外国人監督等の属性の影響を評価。
* **市場価値分析**: 監督効果と初期年俸（対数モデル）や契約年数（負の二項回帰）の相関を検証。
* **頑健性の確認**: `estimatr` を用いたクラスター標準誤差による推定。



## 必要な環境・ライブラリ
分析の再現にはRおよび以下のパッケージが必要です。
- **推定・統計**: `estimatr` (強固な標準誤差), `MASS` / `AER` (過分散検定・負の二項回帰)
- **データ操作・可視化**: `tidyverse` (dplyr, ggplot2)
- **結果出力**: `modelsummary` (テーブル出力), `marginaleffects` (限界効果)

## 実行フロー
1. **データ準備**: `監督効果/` フォルダに必要なパネルデータを配置します。
2. **スコア算出**: `manager_effect.R` を実行し、各監督のスコアを生成します。
3. **分析実行**: `regression.R` を実行し、要因分析の結果を `推定結果/` フォルダに出力します。
